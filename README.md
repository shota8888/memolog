# MemoLog
日々の勉強した内容をメモとして残していくためのサイトです．

# Preview
![demo](https://user-images.githubusercontent.com/59161023/94364477-7935d480-0104-11eb-8161-2e5cec28a9b5.png)

# Requirement
* React 16.13.1
* Next.js 9.5.3
* TypeScript 4.0.2

# Usage
```
$ git clone https://github.com/shota8888/memolog.git
$ cd memolog
$ npm install 
$ npm run dev
```
